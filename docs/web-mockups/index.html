conduit · web mockups

Four directions for the web client. All geometric sans, all denser than today, all in the modern-SaaS minimalism family — with distinct personalities. Each shows the paired session state (most surface area).

01 · quill

Quiet hairlines

Linear-style. Near-monochrome with a single restrained violet accent. 1px borders, no shadows, tight rhythm.

  • Inter · 13.5px body
  • Indigo accent on hover/focus only
  • Mono codes, sans everything else
02 · geist

Pure black & white

Vercel-style. Zero color in chrome — only state pills carry hue. Generous typographic hierarchy, big mono code as hero.

  • Inter + JetBrains Mono
  • Borders or nothing — no fills
  • Code is the focal element
03 · drift

Warm & friendly

Tailscale-style. Warm off-white / deep navy. Teal primary, coral for state. Softly rounded cards, very mild shadow.

  • Cream & navy palette
  • Teal action, coral status
  • 14px rounded corners, light shadow
04 · lume

Subtle premium

Stripe-ish. Cool neutrals, a violet → blue gradient reserved for the primary action, layered hairline depth, careful type scale.

  • Cool gray base
  • Single gradient accent, used sparingly
  • Two-step elevation via borders
